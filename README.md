# The One Day One

A exploration game where discovering your surroundings helps you unlock secrets. 

## Game Concept
**The One Day One** is an exploration game that drops you into an unforgiving environment. To survive, you must explore the wilderness, and carefully gather crucial resources. The world is full of elements to discover, but time and nature are always working against you.

## Controls
Standard PC controls are used for seamless navigation and interaction:

| Key / Action | Function |
| :--- | :--- |
| **W, A, S, D** / **Arrow Keys** | Character Movement |
| **Space Bar** | Jump |
| **Shift** | Run / Sprint |
| **E** | Interact (Gather resources, open containers, etc.) |

## Current Gameplay Objective
The primary goal in the current build is **resource gathering**. 
* Explore your surroundings to locate materials.
* Use the interaction mechanics to harvest resources.

future plans: make the towers interactable to collect special items.
